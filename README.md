# 📖 Overview

Shell scripting is a powerful feature of Linux that allows users to automate repetitive tasks and perform system operations efficiently. This experiment demonstrates the use of **variables, arithmetic operators, conditional statements, loops, and decision-making** through practical Bash programs.

---

# 🎯 Objectives

- Understand Bash scripting fundamentals.
- Use variables and arithmetic operators.
- Implement conditional statements (`if`, `elif`, `else`).
- Use looping constructs (`for`, `while`).
- Develop problem-solving skills using shell scripts.
- Automate common computational tasks.
